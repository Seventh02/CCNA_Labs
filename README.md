# CCNA_Labs

This repository contains Cisco Packet Tracer labs created by me as part of my CCNA studies.

## Technologies Covered (Until now)
- Basic Network Troubleshooting
- IPv4
- IPv6
- Static routing
- Subnetting
- VLANs
    - DTP
    - VTP
- STP/RSTP
- Etherchannel
- Dynamic routing
    - OSPF
- ACLs

## Purpose
These labs demonstrate my understanding of network fundamentals, routing & switching, and basic network troubleshooting using Cisco devices.

Each lab includes:
- Packet Tracer file (.pkt)
- Network topology
- Device configurations
- Lab explanation

## Notes
All labs were designed and configured by me for learning and practice purposes.
