# VLAN and Inter-VLAN Routing Lab

## Description
This lab demonstrates the configuration of a simple network divided into 2 LANs.

## Topology
- 1 Router
- 2 Switches
- 4 PCs

## Technologies Used
- IPv4

## Verification
- PCs in the same VLAN can communicate
- PCs in different VLANs can communicate through the router
