# Network fundamentals

## Description
This lab demonstrates the configuration of a simple network divided into 2 LANs.

## Topology
- 1 Router
- 2 Switches
- 6 EndPoints

## Technologies Used
- IPv4

## Verification
- EndPoints in the same LAN can communicate
- EndPoints in different LANs can communicate through the router

- `Basic_Network_1` → Packet Tracer project
- `Basic_topology_1.png` → Network diagram
