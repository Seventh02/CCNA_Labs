# Network fundamentals

## Description
This lab demonstrates the configuration of a simple network divided into 2 LANs.

## Topology
- 2 Router
- 3 Switches
- 12 EndPoints

## Technologies Used
- Static routing
- Subnetting
  - FLSM
- VLAN
  - Inter-VLAN routing
  - VTP

## Verification
- EndPoints in the same VLAN can communicate
- EndPoints in different VLANs can communicate through the router
